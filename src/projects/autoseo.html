Project Description

The concept of AutoSEO came when we realized we could use ChatGPT to generate cotent for WordPress websites. Initially only an AI content creation plugin, Autoseo has quickly evolved into an all-round automated technical SEO tool.

Developing a WordPress plugin has its own unique challenges, and design constraints. During the design process, I had to make sure what whatever changes we introduco to AutoSEO, it won't break anything in WordPress.

It's been an absolute pleasure to work on this project, putting together the website and the plugin's design and frontend code.
